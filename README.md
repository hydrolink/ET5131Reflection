# Ray'En's Reflections — ET5131

A personal reflection website for ET5131: Venturing into Entrepreneurship at NTU.

## Running locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Editing content

All text on the site lives in one file:

```
content/reflections.ts
```

Edit that file to update any section — no need to touch the components.

## Deploying

Push to GitHub and connect the repository to [Vercel](https://vercel.com). It deploys automatically on every push.
